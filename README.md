## Steps

1. Create a Bot Account in Spark
    * *And make note of the Access Token 
2. Setup yout bot based on the boilerplate
3. Build initial Docker Image and push to Docker Hub 
4. Install Bot on DevNet Sandbox 
5. Test your Bot
